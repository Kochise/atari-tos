  HD_FDC.PRG
  ----------
What it does:	changes the disk drive step rate of TOS 2.06 to 3ms.

Where to put it: inside the AUTO folder of your boot disk or boot partition
of your harddisk.

Why/when do you need it: TOS 2.06 was designed with HD (1.44MB disk drives)
in mind- these run at a 'step rate' of 6ms, which this TOS version defaults
to upon booting. If you're not using a HD disk drive or HD disks an
irritating 'grinding' noise can be heard from the drive. Fortunately this
can be quitened down by using this program- and that's the purpose of it.

The program has been found several places, but with no 'readme' file of it's
own. The German text below has been copied from part of a compilation of
programs.
