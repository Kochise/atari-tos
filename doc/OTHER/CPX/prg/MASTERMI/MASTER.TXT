
Die Regeln dürften bekannt sein... 

Ein '-' in der Bewertung bedeutet, daŞ ein Stein in der gesuchten 
Kombination vorkommt, die Position jedoch nicht stimmt.

Ein '+' in der Bewertung bedeutet, das ein Stein an der richtigen Stelle 
ist.
z.B.:
gesucht:'ABCD' eingegeben: 'AAAA' gibt '+   '
                           'DCBA' gibt '----'
                           'CCAA' gibt '--  '

Werden als Subset Zahlen gewählt, so werden auch Wiederholungen erzeugt. So 
ist z.B. möglich das '1111' der zu entdeckende Code ist.

gesucht '1122' eingegeben: '1111' gibt '++  '
                           '3331' gibt '-   '
                           '2211' gibt '----'

Wird der Code nach einer gewissen Anzahl von Zügen nicht gefunden, so nimmt 
das Modul keine Eingaben mehr an, und die Lösung wird Ihnen für immer 
verborgen bleiben.

Eine neue Kombination erhält man einfach durch Auswählen eines neuen 
Subsets.

Also: Zunächst das gewünschte Subset anwählen [ABCDEF]. Dann im linken Feld 
den 1. Tip durch Anwahl im Stein-Feld (rechts/mitte) und danach Anwahl des 
Feldes, wo der Stein hin soll (unterste Zeile linkes Feld) oder einfaches 
Klicken auf das Zielfeld angeben. Ist der Tip komplet abgegeben (alle 4 
Felder der unteren Zeile besetzt) kann der Computer durch Klicken auf 
[Bewertung] oder auf die Bewertungs-Zeile veranlaŞt werden eine Bewertung 
abzugeben. Und dann heiŞt's nachdenken...
