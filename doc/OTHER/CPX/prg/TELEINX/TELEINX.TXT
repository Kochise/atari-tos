  EINLEITUNG / FUNKTIONSPRINZIP
  ----------------------------------------------------------------------------

  Code CPX, ist ein CPX-Modul, welches mit dem modularen Kontrollfeld von
  ATARI (XCONTROL), benutzt werden kann. Mit TELIINX.CPX kann man Telefon-
  nummern und Adressen auf einfache Art zum Abruf bereit stellen.

  Die Telefonnummern können dazu in einer einfachen ASCII Datei gespeichert
  oder z.B. aus einer bestehenden Datenbank exportiert werden.


  LAUFFÄHIGKEIT
  ----------------------------------------------------------------------------

  TELIINX.CPX sollte dank des modularen Kontrollfeldes von ATARI auf allen
  Rechnern/TOS-Version (>=1.04) und in allen Auflösungen (ST-LOW bis TT-HIGH)
  sowie Grafikkarten laufen.

  TELEINX.CPX benötigt im aktiven Betriebszustand, d.h. wenn es vom Kontroll-
  feld aufgerufen wird oder auch resident geladen wird, ca. 12 KB Speicher und
  je nach Einstellung der max. Einträge ca. 1-20 KB zusätzlichen Daten-
  speicher.

  Im passiven Betriebszustand, d.h. nicht resident im Kontrollfeld und nicht
  gestartete CPX-Application, benötigt TELEINX.CPX nur 512 Bytes (!!!), d.h.
  nur der CPX-Header ist geladen :-).

  INSTALLATION
  ----------------------------------------------------------------------------

  - Kopieren Sie TELEINX.CPX in den CPX-Ordner von XCONTROL (ATARI)
  - Aktivieren Sie das XCONTROLL.ACC, d.h. Endung = .ACC
  - Booten Sie den Rechner neu oder laden Sie im XCONTROLL.ACC unter dem
    Menüpunkt "Voreinstellungen" die CPX-Module neu ein.
  - Rufen Sie TELEINX.CPX auf (Doppelklick auf "Steuercodes" )
  - Klicken Sie auf das PupUp-Menü "Optionen" und wählen Sie den Eintrag
    "SETUP" aus.
  - Klicken Sie auf die Pfadzeile. Es öffenet sich nun eine Fileselectorbox,
    in welcher Sie den Pfad zu der Nummern-Dateie (z.B. TELEBSP.TXT)
    festlegen/einstellen können. Wenn geschehen, klicken Sie "OK" in der
    Fileselectorbox.
  - Und jetzt noch schnell Einstellungen sichern = Klick auf Button "Sichern"
    es wird der Pfad und der Name der Nummern-Datei in TELEINX.CPX gesichtert.
  - Klicken Sie "OK" und sie kommen wieder ins Hauptmenü

  START / BEDIENUNG
  ----------------------------------------------------------------------------

  Nach öffenen des modularen Kontrollfeldes von ATARI, kommen Sie mit
  Doppelklick auf den Eintrag "Tel. Verzeichniss" in das Hauptmenü von
  TELEINX.CPX.

  Hauptmenü
  ---------
  In der Mitte des Fensters sehen sie die Buchstaben A-Z, '?' und '.'.
  Um Telefonnummern abzurufen klicken sie einfach auf den Anfangsbuchstaben
  des gesuchten Namens. Fängt der Name mit einem Umlaut oder Sonderzeichen
  an, so wählen sie '?'. Jetz gelangen sie, wenn unter dem gewählten
  Buchstaben zumindest ein Eintrag gefunden wurde in die:

  Nummern Anzeige
  ---------------
  Hier sehen sie die gefundenen Nummern, linksbündig wird der Name ausgegeben,
  und direkt darunter, rechtsbündig die dazugehörige Telefonnummer. Sind mehr
  als 4 Einträge vorhanden so können Sie mit dem Slider auf der rechten Seite
  durch die Einträge blättern.

  Zurück zum Hauptmenü kommen Sie indem sie die Close-Box links neben der
  Titelzeile "Name/ Nummer" oder die Kopfzeile "Telefonindex V2.xx".

  Klicken Sie auf einen Namen oder eine Telefonnummer, so erscheinten die
  restlichen Daten des Eintrags (z.B. Adresse). Durch nochmaliges klicken
  kommen Sie wieder zurück zur Anzeige der Telefonnummern.


  Setupmenü
  ---------
  Hier haben Sie im oberen Teil die Möglichkeit, die Default-Telefonnummern
  Datei, und die max Anzahl der in der Nummern Anzeige zur Verfügung gestellte
  Nummern festzulegen.

  Unter dem Eintrag "Maximale Einträge je Buchstaben" können sie einstellen
  wieviel für wieviele Einträge das CPX für die Nummern Anzeige bereitstellen
  soll. Reicht der eingestellte Wert nicht aus, so werden nur die max.
  mögliche Anzahl an Nummern geladen und der letzte Eintrag in der Nummern
  Anzeige wird mit "**" markiert.
  Kann das CPX-Modul nicht genügent Speicher für die geforderte Anzahl von
  einträgen bekommen (wenn z.B. ein anderes Programm einfach allen Speicher
  für sich reserviert hat) so erscheint eine Allert-Box. Eine anzeige der
  Nummer ist dann nicht möglich.


  Etwas weiter unten, können Sie durch Klick auf die dort angezeigte
  Pfadzeile, die Telefonnummern Datei auswählen.

  Durch Anwahl von "Sichern" werden die vorgenommenen Einstellungen dauerhaft
  in TELEINX.CPX gespeichert.

  Wählen Sie "Abbruch", so werden die geänderte Daten nicht übenommen, Sie
  arbeiten dann mit der Einstellung weiter, die Sie vor'm aufruf des Setup
  Menüs hatten.

  Verlassen sie das Setup Menü mit "OK" so werden die gemachten Änderungen
  ubernommen und bleiben bis zum Verlassen des CPX-Moduls aktiv.


  DAS FORMAT DER TELEFONNUMMERN DATEI
  ----------------------------------------------------------------------------

  Die Telefonnummern Datei ist denkbar einfach aufgebaut, ein erstellen per
  Editor oder als Export aus einem Datenbank-Programm sollte keine Probleme
  bereiten.

  Die einzelnen Datenfelder werden durch '|' getrennt, die Datensätze durch
  CR LF (Zeilenwechsel).

  Z.B.:
  Mandel Andreas|Fax:0781/38807|Postfach 26|7601 Ortenberg|<77799> Ortenberg

  Das erste Feld sollte immer den Namen enthalten, im 2. Feld sollte die
  Telefonnummer stehen. Den Inhalt der folgenden Felder können sie Frei
  bestimmen.

  Mehrfache Leerzeichen werden beim Einlesen der Datei zu einem Zusammen-
  gefa×t.

  Ein Datenfeld sollte nicht mehr als 28 Zeichen enthalten. Längere
  Datenfelder werden entsprechend gekürzt.

  Pro Datensatz sind max 8 Felder möglich, folgen weitere, so werden sie
  ignoriert.
