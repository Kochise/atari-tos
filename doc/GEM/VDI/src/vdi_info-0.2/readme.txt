vdi_info-0.2

This is a little program that display VDI information. You can redirect
output to file 'stdout.txt' with '-file' parameter. It seems it does not
work under TOS 4, so use bash for example to redirect the standard output
to a file.

Changes 0.1 -> 0.2
- Corrected bug when reading bit organization of True Colour pixels.
- Added display of masks
